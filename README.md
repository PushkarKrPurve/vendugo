# Vendugo – Car Rental Website

**Vendugo** is a modern car rental website built with **HTML** and **Tailwind CSS**.  
The site features a clean, responsive design with an interactive experience for users looking to rent off-road and adventure vehicles.

**🌐 Live Demo:** [Vendugo Live](https://pushkarkrpurve.github.io/vendugo/)

---

## Features

- **Responsive design** – works perfectly on desktop, tablet, and mobile devices.  
- **Auto-sliding image gallery** – showcase vehicles or highlights with smooth automatic sliders.  
- **Built with Tailwind CSS** – clean, modern styling without writing extensive custom CSS.  
- **Interactive UI elements** – hover effects, buttons, and navigation designed for usability.  
- **Easy-to-navigate structure** – clear sections for browsing content.  

---

## Built With

- [HTML5](https://developer.mozilla.org/en-US/docs/Web/Guide/HTML/HTML5)  
- [Tailwind CSS](https://tailwindcss.com/)  
- Vanilla JavaScript (for auto-sliding gallery and interactivity)
